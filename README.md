# CodeAlpha-TodoList
The To-Do List App is a simple application that allows users to manage their tasks efficiently. Users can create notes with titles, edit them as needed, and delete them when they are no longer required.
## Features
  - Create Notes
  - Edit Notes
  - Delete Notes
  - Responsive
